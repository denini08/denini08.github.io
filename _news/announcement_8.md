Great news: our tool demo paper on runtime verification for Python has been **accepted at FSE 2026**! 🎉

---

I'm thrilled to announce that our paper titled _"PyMOP: A Runtime Verification Tool for Python"_ has been accepted at the **ACM International Conference on the Foundations of Software Engineering (FSE 2026)**, in the Tool Demonstrations track. The conference will be held in Montreal, Canada, in July 2026.

This work was developed in collaboration with Zhuohang Shen, Mohammed Yaseen, Kevin Guan, my supervisor Marcelo d'Amorim, and Owolabi Legunsen. PyMOP is a generic, extensible, and efficient runtime verification tool for Python that supports five specification logics, five monitoring algorithms, and ships with 81 API specifications for Python and widely-used libraries. Evaluated on 48,112 unit tests across 847 GitHub projects, PyMOP is up to 419× faster than recent dynamic analysis tools — and 75 of the 156 bugs it found have already been fixed by developers.
